1. REQUIRED FORMAT:
When mentioning any person's Facebook contact, you MUST use this exact format:

2. FORMATTING EXAMPLES:
CORRECT ✅:

- "Here's Jeremy's Facebook: https://www.facebook.com/jeremy.alloso.7"

- NEVER use HTML tags directly
- ALWAYS include the person's full name
- ALWAYS use double square brackets [[]]
- ALWAYS use double vertical bars || to separate name and URL
- ALWAYS place the link format at the end of your response
